# Privacy Policy

**Adapt Bot**
Last updated: July 1, 2026

---

## 1. Overview

This Privacy Policy explains what data Adapt ("the Bot") collects, how it is used, and how it is stored. By using the Bot, you agree to the collection and use of data as described here.

---

## 2. Data We Collect

Adapt collects and stores the following data in order to function:

### Per Server (Guild)
| Data | Purpose |
|---|---|
| Server ID | Identifying server-specific settings |
| Channel IDs | Welcome, logging, ticket, mod log channels |
| Role IDs | Mute role, auto-roles, support roles |
| Custom settings | Prefix, leveling config, economy config, automod rules |

### Per User
| Data | Purpose |
|---|---|
| User ID | Identifying users across all features |
| XP & Level | Leveling system |
| Coin balance | Economy system |
| Warnings | Moderation logs |
| Moderation history | Mod action logs |
| Inventory | Economy shop purchases |
| Giveaway entries | Giveaway participation |
| Trivia scores | Trivia statistics |

### Message Content
- **Automod:** Message content is temporarily scanned for spam, links, and bad words. It is **not permanently stored** unless a violation is logged.
- **Custom commands:** Command responses are stored as configured by server admins.
- **Deleted/edited messages:** Content may be logged to a server's log channel if logging is enabled by the server admin.

---

## 3. How We Use Your Data

Data is used **only** to provide the Bot's features:

- Delivering server-specific configuration
- Tracking XP, levels, and economy balances
- Recording moderation actions
- Running giveaways and ticket systems
- Generating logs for server administrators

We do **not**:
- Sell your data to third parties
- Use your data for advertising
- Share your data with anyone outside the Bot's operation

---

## 4. Data Storage

All data is stored securely in a **Supabase PostgreSQL database**. Access is restricted to the Bot's backend only. Data is retained as long as the Bot remains in your server or until you request deletion.

---

## 5. Data Deletion

You may request deletion of your data at any time:

- **Server admins** can request full server data deletion by removing the Bot and contacting us
- **Individual users** can request deletion of their personal data by contacting the Bot owner

To request deletion, open an issue on the [GitHub repository](https://github.com/ajwadnxt/adapt-bot) or contact the Bot owner on Discord.
